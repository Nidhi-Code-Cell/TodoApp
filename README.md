# To-Do List Web Application

A simple and interactive To-Do List web application that allows users to add and delete tasks. Tasks are stored using the browserâ€™s localStorage, ensuring data persistence even after page refresh or browser restart until the user deletes them.

## Features
- Add new tasks
- Delete existing tasks
- Persistent task storage using localStorage
- Tasks remain saved after refresh or app close
- Clean and responsive user interface

## Tech Stack
- HTML
- CSS
- JavaScript
- Browser localStorage

## How It Works
1. User adds a task through the input field
2. Task is saved in localStorage
3. On page load, tasks are retrieved from localStorage
4. User can delete tasks, which updates localStorage accordingly
